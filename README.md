# Loro_Piana_DPP

## Overview
This project contains a **prototype of an immersive Digital Product Passport** for Loro Piana, combining compliance, traceability, and brand storytelling.  

## Objectives
- Demonstrate a premium, user-friendly DPP experience  
- Include sustainability, authenticity, and traceability features  
- Align with ESPR/ESRS regulations and LVMH LIFE 360 roadmap
